🚀 Jornada Python: Do Zero ao Avançado

Este repositório foi criado com o objetivo de documentar minha evolução no aprendizado de Python, começando do básico até conceitos mais avançados, incluindo também noções de banco de dados.

Aqui você vai encontrar exercícios, exemplos práticos e pequenos projetos desenvolvidos ao longo dos meus estudos.

📚 Sobre o projeto

Durante minha jornada, venho explorando:

Fundamentos da linguagem Python
Estruturas de controle (if, else, loops)
Funções e modularização
Estruturas de dados (listas, tuplas, dicionários)
Programação orientada a objetos (POO)
Manipulação de arquivos
Integração com banco de dados

O objetivo principal é consolidar o conhecimento na prática e acompanhar minha evolução ao longo do tempo.
